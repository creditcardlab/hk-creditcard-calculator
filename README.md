# 💳 信用卡管家 Credit Card Manager

一個幫你計算同追蹤信用卡回贈嘅智能工具。

## ✨ 主要功能

- 🧮 **智能計算器** - 輸入金額同類別，即時比較所有卡片回贈
- 📊 **進度追蹤** - 監控月簽任務、回贈額度、推廣活動進度
- 💾 **數據備份** - 匯出/匯入功能，保障你嘅數據安全
- 🎯 **額度監控** - 自動提醒Cap位，避免「爆Cap」

## 🚀 使用方法

1. 開啟網頁後，去「設定」頁面揀選你有嘅信用卡
2. 喺「計算」頁面輸入簽賬金額同類別
3. 系統會自動排序，顯示最佳回贈嘅卡
4. 點擊「+ 記一筆」就可以記錄交易

## 💡 小貼士

- 定期喺「設定」匯出數據作備份
- 留意「進度」頁面嘅任務提醒
- 用「🏆 最佳」標籤快速識別最抵卡

## 📱 支援平台

- ✅ 手機瀏覽器
- ✅ 電腦瀏覽器
- ✅ 平板

## 🔒 私隱

所有數據儲存喺你嘅瀏覽器本地（localStorage），唔會上傳去任何伺服器。
